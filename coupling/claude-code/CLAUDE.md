# Learn Commands

## Commands

### /learn {correction}

Capture a correction or insight.

1. Classify:
   - Session-only → Apply now
   - Standing preference → Add to this file or learnings
   - Skill → Document in .claude/learnings.md
   - Domain knowledge → Add to relevant context file

2. Acknowledge what was learned
3. Apply immediately
4. Persist if appropriate

### /postmortem

After a failure, analyze:

1. **What happened:** Facts only
2. **Root cause:** The actual why
3. **Mitigation:** Concrete fix (required)
4. **Pattern:** Generalizable learning

Every postmortem needs a real mitigation.
