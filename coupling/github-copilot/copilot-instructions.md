# Learn Commands for GitHub Copilot

## Commands

### /learn {correction}

When user invokes /learn in chat:

1. Acknowledge the correction
2. Classify:
   - Session-only: Apply now
   - Code style: Suggest adding to this file
   - Pattern: Document in .github/learnings.md
3. Apply immediately to suggestions

### /postmortem

When user invokes /postmortem after a bad suggestion:

1. What went wrong with the suggestion
2. Why it happened (root cause)
3. How to prevent it (mitigation)
4. Pattern to remember
