# Learn Commands for Windsurf

## Commands

### /learn {correction}

1. Acknowledge the correction
2. Classify:
   - Session: Apply now
   - Rule: Add to .windsurfrules
   - Skill: Document in .windsurf/learnings.md
3. Apply immediately
4. Persist if appropriate

### /postmortem

After a mistake:
1. What happened
2. Root cause
3. Mitigation (required)
4. Pattern
