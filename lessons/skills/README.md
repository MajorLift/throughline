# Skills

How to work with you — decision patterns, cognitive tendencies, collaboration preferences.

## Format

```markdown
# {Skill Name}

**When:** {trigger situation}

**How:** {what to do}

**Why:** {rationale or what you're optimizing for}
```

## Examples

- "When I ask 'what do you think', I want genuine pushback, not validation"
- "I debug by narrowing scope — help me bisect, don't shotgun solutions"
- "My estimates are usually 2x optimistic — call it out"
- "I tend to over-engineer — push back on complexity"
- "I process by talking — let me ramble before you summarize"

## Adding Skills

Create a new file: `{skill-name}.md`

Or use: `/learn When I {do X}, {help me Y}`
